# AXOMED Prototype 1

AXOMED is an IoT-based patient monitoring prototype designed to track vital signs in real time.

## Features
- Heart Rate Monitoring (BPM)
- Body Temperature Monitoring
- Room Temperature & Humidity
- Real-time web dashboard
- ESP32 → Node.js → Web Interface

## Hardware
- ESP32
- MAX30102 Heart Rate Sensor
- DS18B20 Body Temperature Sensor
- DHT11 Sensor
- OLED Display

## Software
- Arduino IDE
- Node.js
- HTML/CSS/JavaScript

## Architecture
ESP32 → WiFi → Node.js Server → Web Dashboard

## Purpose
Built as a prototype for healthcare monitoring systems and startup validation.
